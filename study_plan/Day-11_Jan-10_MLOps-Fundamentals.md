## Key Terms to Know

| Term | Definition |
|------|------------|
| **MLOps** | Practices for deploying and maintaining ML models in production |
| **Feature Store** | Centralized repository for ML features |
| **Model Registry** | Tool for versioning and managing trained models |
| **Feature Group** | Collection of related features |
| **Feature View** | Configured view of features for serving |
